# Issue Tracker

Issues for this repo are tracked in **Jira**, project key **`WC`**, accessed via the **Jira MCP server**.

## How to create/update issues

Use the Jira MCP server tools (e.g. `mcp__jira__create_issue`, `mcp__jira__update_issue`). Do not use `gh issue create` for this repo.

## Project details

- Project key: `WC`
- Access: Jira MCP server (configured in Claude Code MCP settings)

## Notes

If the MCP connection fails, stop and tell the user — do not attempt workarounds or guess from cached data.
